### Prerequisites

- Node.js installed
- MongoDB database (local or Atlas)

### Installation Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/akonBRO/ClubSync.git
   cd ClubSync
   ```
2. Install dependencies:

   ```bash
   cd backend
   npm install
   cd ../frontend
   npm install
   ```

    
3. Run the application:

    ```bash
    # Start backend
    cd backend
    npm start
    
    # Start frontend
    cd ../frontend
    npm start
    App runs on http://localhost:3000
    ```
